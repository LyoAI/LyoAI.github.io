Astra: Activation-Space Tail-Eigenvector Low-Rank Adaptation of Large Language Models
Kainan Liu, et al.
We propose Astra, a novel low-rank adaptation method that operates in the activation space of LLMs. By identifying and fine-tuning along tail eigenvectors of the activation covariance matrix, Astra achieves superior parameter efficiency and task performance compared to standard LoRA approaches, with minimal inference overhead.
Paper Code

Key Methodology

Astra operates by identifying tail eigenvectors in the activation covariance matrix of large language models, then only fine-tuning along these directions. This approach significantly reduces the number of trainable parameters while maintaining or improving performance across various downstream tasks.

Key Results

  • Outperforms standard LoRA by 3-5% on 8 out of 12 NLP benchmarks
  • Uses 40% fewer parameters than comparable LoRA approaches
  • No inference overhead compared to full fine-tuning

GRASP: Replace Redundant Layers with Adaptive Singular Parameters for Efficient Model Compression
Kainan Liu, et al.
We introduce GRASP, a model compression framework that identifies and replaces redundant transformer layers with learnable singular parameters. This approach preserves model performance while significantly reducing parameters and computational cost, enabling efficient deployment of large language models in resource-constrained environments.
Paper Code

Key Methodology

GRASP (Gradient-based Redundancy-Aware Structured Pruning) identifies redundant transformer layers by analyzing gradient flow during fine-tuning, then replaces them with lightweight singular value decomposition (SVD) projections. This approach maintains model performance while significantly reducing inference latency and memory usage.

Key Results

  • Reduces model size by 50% with less than 1% performance degradation
  • Improves inference speed by 2.3x on commodity hardware
  • Outperforms existing pruning methods by 12% on average across NLP tasks

Detecting and dissecting anomalous anatomic regions in spatial transcriptomics with STANDS
..., Kainan Liu, et al.
STANDS is a computational framework for detecting and dissecting anomalous anatomic regions in spatial transcriptomics data. It integrates spatial information and gene expression profiles to identify tissue regions with abnormal molecular signatures, providing insights into disease pathology and tissue heterogeneity at unprecedented resolution.
Paper Code

Key Methodology

STANDS (Spatial Transcriptomics Anomaly Detection and Segmentation) combines graph neural networks with statistical modeling to identify anomalous tissue regions in spatial transcriptomics data. It leverages both gene expression profiles and spatial neighborhood information to detect molecular signatures associated with disease pathology.

Key Results

  • Achieves 94% accuracy in detecting tumor regions in breast cancer spatial transcriptomics data
  • Identifies novel disease-associated cell subpopulations with spatial localization
  • Outperforms existing methods by 18% on spatial anomaly detection benchmarks
